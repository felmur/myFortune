# myFortune
"Fortune" server with a lot of quotes, written in pure c++.

# Compilation
$> cmake .

$> make

# Installation
Manually copy executable 'myFortune' into your /usr/bin

# Usage
myFortune [port | -h]

If you don't specify any arguments, the default port will be 60060.

# Real Test
$> nc localhost 60060
